River Places Tool

This tool uses Google Places to take a list of cities and a list of keywords and return a list of related places in those cities.

List of locations

Separate each location on a new line. Order is unimportant. Duplicate places that are returned for more than one city will be removed. Specify unique locations; for instance using both city and state.

List of keywords/phrases

Separate each keyword/phrase on a new line. Priority will be assigned based on which keyword matched first. Optionally, add the priority after the keyword. Duplicate places will be removed.

Fields

Select the fields you wish to return.


Places Data

Date returned from Google Places API in tab-separated values format. This should be possible to paste into Excel or Google docs. Note the header row which designates the fields.
